# JusticePath

**Rehabilitation & Innocence Support Platform**

A comprehensive mobile and web application designed to help individuals navigate the criminal justice system — whether working toward rehabilitation, proving innocence, or rebuilding life after incarceration.

## Mission

Provide accessible legal tools, rehabilitation resources, and reentry support to people who need it most — especially those who can't afford traditional legal representation.

## Core Features

### 🔍 Case Tracker
- Case status dashboard with timeline visualization
- Court date reminders and notifications
- Document organizer for legal paperwork
- Evidence logging and photo/video capture

### ⚖️ Legal Support
- AI-powered legal Q&A (plain language explanations)
- Lawyer directory with filters (public defenders, pro bono, specialty)
- Document templates (motions, appeals, expungement petitions)
- Rights education by jurisdiction

### 📚 Rehabilitation Pathway
- Personalized rehabilitation plan builder
- Educational course tracking (GED, vocational certs)
- Mental health resources and crisis support
- Substance abuse program tracker
- Community service hour logging

### 🆕 Innocence Support
- Evidence organization tools
- Witness statement collection forms
- Innocence project directory (national registry)
- DNA testing resource guide
- Appeal pathway builder

### 🏠 Reentry Support
- Housing resource finder
- Employment resources and job boards
- ID/document replacement guide
- Benefits enrollment assistance (SNAP, Medicaid, etc.)
- Financial literacy and credit rebuilding

### 🤝 Community & Mentorship
- Anonymous peer support forums
- Mentor matching (post-release success stories)
- Family support resources
- Faith-based and secular counseling directories

## Tech Stack

- **Frontend:** React Native (iOS + Android) + React Web
- **Backend:** Node.js + Express
- **Database:** PostgreSQL
- **Authentication:** JWT + optional phone-based auth (no email required)
- **AI:** Open-source LLM for legal Q&A (local-first when possible)
- **Hosting:** Self-hosted or privacy-focused cloud

## Privacy & Security

- End-to-end encryption for sensitive documents
- Zero-knowledge architecture — we don't sell data
- Minimal data collection — users own their data
- Offline mode for core features
- Panic button / quick-exit for safety situations

## Getting Started

```bash
# Clone the repository
git clone https://github.com/[YOUR_USERNAME]/JusticePath.git
cd JusticePath

# Install dependencies
npm install

# Set up environment
cp .env.example .env

# Run development server
npm run dev
```
